Privacy Policy

Effective Date: March 15, 2026

Magic Drag ("Service") values your privacy and complies with applicable data protection laws. This policy explains what information we collect, how we use it, and how we protect it.

1. Information We Collect

a. When you sign up

  • Email address — via Google sign-in

b. Automatically collected during use

  • IP address — to prevent abuse
  • Selected text — input for AI reply generation
  • AI-generated replies — for quality improvement and error analysis
  • Usage count and timestamps — to manage free/paid plan limits
  • User settings (tone, language, persona) — to generate personalized replies

c. Guest (non-member) use

  • Randomly generated identifier (UUID) — to manage free trial limits
  • IP address — to prevent abuse

2. How We Use Your Information

We use collected information only for the following purposes:

  • Service delivery: Generate and deliver AI-based replies
  • Usage management: Track daily usage limits for free and paid plans
  • Service improvement: Analyze reply quality and diagnose errors
  • Abuse prevention: Detect and block abnormal usage via IP
  • Account management: Authentication and login session maintenance

3. Third-Party Data Sharing

We do not share your personal information with third parties, except as follows:

Google (Gemini API)

  • Data shared: Selected text, user settings
  • Purpose: AI reply generation
  • Note: We use the Google Gemini API in a way that your data is not used to train Google's global models.

Supabase (Cloud infrastructure)

  • Data shared: Email, usage records
  • Purpose: Data storage and authentication

We may also disclose information when required by law.

4. Data Retention and Deletion

  • Account information (email): Retained until account deletion. Deleted immediately upon request.
  • Usage records: Retained for 90 days from collection. Automatically deleted after 90 days.
  • IP address: Retained for 1 hour. Automatically deleted after 1 hour.
  • Guest identifier: Retained for 30 days from last use. Automatically deleted after 30 days.

5. Your Rights

You may exercise the following rights at any time:

  • Request access to your personal information
  • Request correction of inaccurate information
  • Request deletion of your information
  • Delete your account by contacting us via email

6. Browser Permissions

Magic Drag uses the following browser permissions:

  • activeTab — Detect selected text on the current tab
  • scripting — Display reply UI on web pages
  • storage — Store login info and settings locally
  • identity — Google sign-in

7. Security Measures

  • All data is transmitted over HTTPS (TLS) encryption
  • Authentication tokens are stored securely in the browser's local storage
  • Server-side data is protected by Supabase Row Level Security (RLS)
  • Passwords are hashed; plaintext is never stored

8. Changes to This Policy

Any changes to this policy will be announced through the extension or our official page.

1. 수집하는 개인정보 항목

가. 회원가입 시

  • 이메일 주소 — Google 소셜 로그인

나. 서비스 이용 중 자동 수집

  • IP 주소 — 비정상 이용 방지(남용 탐지)
  • 사용자가 선택한 텍스트 — AI 답변 생성을 위한 입력값
  • AI 생성 답변 — 서비스 품질 개선 및 오류 분석
  • 서비스 이용 횟수·일시 — 무료/유료 플랜 사용량 관리
  • 사용자 설정값 (톤, 언어, 페르소나) — 맞춤형 답변 생성

다. 비회원(게스트) 이용 시

  • 임의 생성 식별자 (UUID) — 무료 체험 횟수 관리
  • IP 주소 — 비정상 이용 방지

2. 개인정보의 이용 목적

수집한 정보는 다음 목적으로만 이용합니다.

  • 서비스 제공: AI 기반 답변 생성 및 결과 전달
  • 사용량 관리: 무료·유료 플랜별 일일 사용 횟수 관리
  • 서비스 개선: 답변 품질 분석 및 오류 진단
  • 남용 방지: IP 기반 비정상 이용 탐지 및 차단
  • 계정 관리: 회원 인증, 로그인 상태 유지

3. 개인정보의 제3자 제공

서비스는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다만, 다음의 경우 예외로 합니다.

Google (Gemini API)

  • 제공 항목: 사용자가 선택한 텍스트, 설정값
  • 제공 목적: AI 답변 생성
  • 참고: 전송된 텍스트 데이터는 Google의 AI 모델을 학습시키는 데 활용되지 않습니다.

Supabase (클라우드 인프라)

  • 제공 항목: 이메일, 사용 기록
  • 제공 목적: 데이터 저장 및 인증

법령에 의한 요청이 있는 경우 관련 절차에 따라 제공할 수 있습니다.

4. 개인정보의 보유 및 파기

  • 회원 정보 (이메일): 회원 탈퇴 시까지 보유. 탈퇴 요청 후 즉시 삭제.
  • 서비스 이용 기록: 수집일로부터 90일 보유. 90일 경과 후 자동 삭제.
  • IP 주소: 수집일로부터 1시간 보유. 1시간 경과 후 자동 삭제.
  • 게스트 식별자: 최종 이용일로부터 30일 보유. 30일 경과 후 자동 삭제.

5. 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 열람 요청: 본인의 개인정보 처리 현황을 확인할 수 있습니다.
  • 수정 요청: 잘못된 개인정보의 정정을 요청할 수 있습니다.
  • 삭제 요청: 개인정보의 삭제를 요청할 수 있습니다.
  • 회원 탈퇴: 이메일로 탈퇴를 요청할 수 있습니다.

6. 브라우저 권한 안내

Magic Drag 확장 프로그램은 다음 브라우저 권한을 사용합니다.

  • activeTab — 현재 탭에서 선택한 텍스트 감지
  • scripting — 웹페이지에 답변 UI 표시
  • storage — 로그인 정보 및 설정값 로컬 저장
  • identity — Google 소셜 로그인

7. 개인정보 보호를 위한 기술적 조치

  • 모든 데이터 전송은 HTTPS(TLS) 암호화를 적용합니다.
  • 인증 토큰은 브라우저 로컬 저장소에 보관됩니다.
  • 서버 측 데이터는 Supabase의 Row Level Security(RLS) 정책으로 보호됩니다.
  • 비밀번호는 해시 처리되어 저장되며, 원문은 보관하지 않습니다.

8. 개인정보 처리방침의 변경

본 방침이 변경되는 경우, 변경 사항을 확장 프로그램 내 또는 공식 페이지를 통해 공지합니다.
